# BMI Hesaplama

Bu proje, kullanıcıdan alınan kilo ve boy bilgilerine göre vücut kitle indeksini (BMI) hesaplayan basit bir Python uygulamasıdır.

## Özellikler
- Kullanıcıdan kilo ve boy bilgisi alır
- BMI (Vücut Kitle İndeksi) hesaplar
- Sonuca göre kullanıcıyı sınıflandırır (zayıf, normal, fazla kilolu, obez)

## BMI Formülü
BMI = kilo / (boy * boy)

## Kullanılan Teknoloji
- Python
